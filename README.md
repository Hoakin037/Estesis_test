## Руководоство по проекту

### Установка зависимостей
Проект разрабатывался и тестировался на версии Python 3.10.0
1. Зайдите в директорию проекта и создайте виртуальное пространство ```python -m venv .venv``` 
2. Перейдите в созданное пространство через ```.venv/scripts/activate```
> Для Linux ```source .venv/bin/activate```
3. Установите все  зависимости через ```pip install -r requirements.txt```
4. Создайте пустую бд Postrgresql с любым названием
5. Создайте .env в корне директории по .env.example

### Запуск приложения

```uvicorn main:app --host 0.0.0.0 --port 8080 --reload```
